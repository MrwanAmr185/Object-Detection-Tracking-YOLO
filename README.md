# Object Detection and Tracking

A real-time object detection and tracking app using YOLO and OpenCV

## Features
- Real-time object detection using webcam
- Draws bounding boxes around detected objects
- Tracks objects and assigns unique IDs
- Supports 80+ object classes

## Technologies Used
- Python
- OpenCV
- Ultralytics YOLOv11

## Installation

1. Clone the repository
git clone https://github.com/MrwanAmr185/Object-Detection-Tracking-YOLO

2. Install dependencies
pip install opencv-python ultralytics

3. Run the app
python code.py

## Usage
1. Run the script
2. Allow camera access
3. Objects will be detected and tracked in real time
4. Press q to quit

## Note
The YOLO model file will be downloaded automatically the first time you run the script. No need to download it manually
